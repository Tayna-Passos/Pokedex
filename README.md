# 📱 Pokédex Interativa

![Pokemon Header]:(https://tayna-passos.github.io/Pokedex/)

Uma aplicação web leve e dinâmica que permite navegar pelo universo Pokémon. O projeto consome dados da [PokeAPI](https://pokeapi.co/) e foi planeado para oferecer uma experiência fluida em qualquer dispositivo.

## 🔗 Link do Projeto
Acesse aqui:(https://tayna-passos.github.io/Pokedex/)

---

## 📱 Experiência Mobile-First
Este projeto foi desenvolvido com foco na **responsividade**. 
* **Adaptabilidade:** O layout ajusta-se automaticamente para ecrãs de smartphones, tablets e desktops.
* **Navegação por Toque:** Elementos e botões dimensionados para uma interação fácil em dispositivos móveis.
* **Performance:** Carregamento otimizado para não sobrecarregar dados móveis.

## 🚀 Funcionalidades Principais
- **Consumo de API:** Requisições assíncronas para obter dados em tempo real.
- **Busca Dinâmica:** Encontre o seu Pokémon favorito instantaneamente.
- **Cartões Detalhados:** Exibição de tipos, IDs e artes oficiais.
- **Scroll Infinito/Botão Load More:** Explore a lista completa sem perder performance.

## 🛠️ Tecnologias
- **HTML5** & **CSS3** (Variáveis CSS e Flexbox/Grid)
- **JavaScript (ES6+)** (Fetch API, Async/Await)
- **PokeAPI**
